## Shared assets

- `main.js` — mobile nav toggle + scroll-reveal animations

## Hosting

Static site on GitHub Pages — no build step, no dependencies.

1. All files live in the repo root (same folder, exact lowercase names).
2. **Settings → Pages → Deploy from a branch → `main` / `/ (root)`.**
3. GitHub serves `index.html` as the homepage; relative links handle the rest.

To edit content, open the relevant `.html` file directly. Shared styling lives in `style.css`.
